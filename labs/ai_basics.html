AI Basics

Other formats: Markdown version

In this tutorial, we explore how AI assistants can help with data visualization tasks. We'll learn how to effectively prompt AI to generate code and understand the importance of human oversight.

Introduction

AI assistants like ChatGPT, Claude, and GitHub Copilot can help with visualization tasks. Learn how to use them effectively while maintaining critical thinking and design expertise.

Effective Prompting

Good prompts are specific, provide context, and clarify expectations. Learn techniques for getting better results from AI assistants.

Iteration

AI rarely gets it perfect on the first try. Learn how to iterate on AI-generated code and refine visualizations.

Human Oversight

AI can make mistakes and produce misleading visualizations. Learn how to critically evaluate AI output and apply design principles.
