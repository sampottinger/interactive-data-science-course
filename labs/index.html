Skills Labs

Hello! This page provides links to all of our skills labs for the semester. It will be updated as we go along. Remember that Skills Labs use a "flipped classroom" so you will complete these on your own time. However, I will be available during regular lecture hours and office hours to assist. Look forward to seeing what you make! Start first tutorial.

Lab 1: Hello Python

This skills lab introduces Python fundamentals. This corresponds to Lesson 3.
  • Tutorial 1: Python Introduction

Lab 2: Expanding Python

These skills labs explore creative Python programming and object-oriented concepts. This corresponds to Lesson 4.
  • Tutorial 2: Python Graphics
  • Tutorial 3: Python Objects

Lab 3: Glyphs

These skills labs explore how to manipulate glyphs and their properties, taking advantage of preattentive features. We will also briefly look at other basic graphical elements (text and images) through Sketchingpy. This corresponds to Lesson 8.
  • Tutorial 4: Forms
  • Tutorial 5: Color and Style
  • Tutorial 6: Position
  • Tutorial 7: Orientation

Lab 4: Outside

Now that we have drawing down, we will look at what from the outside world we can bring into our work, including the user and data. This corresponds to Lesson 9.
  • Tutorial 8: Text
  • Tutorial 9: Images and Buffers
  • Tutorial 10: User Loops
  • Tutorial 11: Data

Lab 5: JavaScript

We finish up by looking beyond Python. Though optional, I have some resources for you to consider if you want to continue your journey into other technologies.
  • Tutorial 12: JS

Lab 6: AI

This skills lab explores how to use AI assistants for data visualization, emphasizing the importance of human design expertise in guiding AI to create effective visualizations. This corresponds to Lesson 26.
  • Tutorial 13: AI Basics
  • Tutorial 14: AI Advanced
